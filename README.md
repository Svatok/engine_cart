## Installation
1. Add to your gemfile: gem 'engine_cart', github: 'Svatok/engine_cart', branch: 'dev'
2. Run bundle install
3. Run rails g corzinus:install USER_MODEL PRODUCT_MODEL

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install engine_cart
```
