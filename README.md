## Installation
1. Add to your gemfile:
```ruby
gem 'engine_cart', github: 'Svatok/engine_cart', branch: 'dev'
```
2. Execute:
```bash
$ bundle install
```
3. Install EngineCart: 
```bash
$ rails g engine_cart:install USER_MODEL PRODUCT_MODEL
```
